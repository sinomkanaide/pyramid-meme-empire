# Share Card Backgrounds

Place 3 background images here for the share card system:

- `card-1.webp` — Background option 1 (600x800px recommended)
- `card-2.webp` — Background option 2 (600x800px recommended)
- `card-3.webp` — Background option 3 (600x800px recommended)

If images are missing, the share card will use a dark gradient fallback.
